NOMENCLATURA

Nombre De Identificadores:

Los nombres de los identificadores pueden comenzar con:

  • Una letra
  • Un signo de dolar $
  • Un guión bajo _
  • Núnca con números o caracteres especiales

Nombrar Archivos

Para nombrar archivos en JavaScript se usa snake_case. Ejemplo:

main_content.js

Nombrar Constantes

Para nombrar constantes en JavaScript se usa UPPE_SNAKE_CASE. Ejemplo:


const UNA_CONSTANTE = 3.1416;

Nombrar Clases

Para nombrar clases en JavaScript se usa UpperCamelCase. Ejemplo:


class UnaClase {
    constructor() {
        this.nombre = nombre;
        this.edad = edad;
    }

    unSaludo() {
        return `Hola, ${this.nombre}.`;
    }
};

Nombrar Variables, Objetos, Funciones E Instancias De Clase

Para nombrar variables objetos, funciones e instancias de clase, se utiliza lowerCamelCase. Ejemplo:


let unaVariable = "Hola";

const unObjeto = {
    nombre: "Guillermo",
    edad: 24
};

function unaFuncion() {
    console.log("Hola mundo");
};
unaFuncion();

const unaInstancia = new UnaClase();
