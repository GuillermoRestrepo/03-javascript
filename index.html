INTRODUCIÓN DE JavaScript

JavaScript es un lenguaje de programación que se utiliza principalmente para crear contenido interactivo en sitios web. Es uno de los tres pilares del desarrollo web, junto con:

  • HTML - (estructura del contenido)
  • CSS - (estilo y diseño)
  • JavaScript - (comportamiento e interactividad)

¿Dónde Se Ejecuta?

Principalmente, JavaScript se ejecuta en el navegador del usuario, pero también puede ejecutarse en el servidor gracias a tecnologías como Node.js.

En El Navegador

En La Consola Del Navegador

Para esto se necesita abrir las herramientas de desarrollador del navegador y dar clic en la pestaña de consola.

Se desplegará una mini terminal, en la cual se podrá ejecutar código de JavaScript.

Incluido En HTML

Para esto, se utiliza la etiqueta <script> dentro de un documento HTML:


                <body>
                    <script>
                        console.log("Hola desde un documento HTML.")
                    </script>
                </body>
En Un Archivo Externo

Para esto, hay q crear un archivo con extención (.js) y vincularlo al documento HTML, como se muesta a continuación:

En (archivo.js):


                console.log("Hola desde un archivo externo.")

En HTML:


                <body>
                    <script src="archivo.js"> </script>
                </body>

Usando Node.js

Se instala Node.js, luego se crea un archivo con extención (.js) y después es ejecutado desde una terminal:

En (archivo.js):


        console.log("Hola desde JavaScript.")

En la terminal:


        node archivo.js
